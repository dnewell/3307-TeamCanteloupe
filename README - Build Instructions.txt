Supported platforms:

- Windows 7/8/8.1/10 - currently supported

- OSX - planned, not currently supported


Build Requirements:

- Qt Creator 5.5 or above, with mingw compiler suite (included in default Qt install)


Build Process:

1) open 3307_GUI.pro project file in Qt Creator

2) click 'yes' to the warning about user.pro files

3) on 'Configure Project' screen, click 'Configure Project'

4) in Build menu, click Run QMake

5) in Build menu, click build (or CTRL-B)

6) in Build menu, click Run (or CTRL-R)
