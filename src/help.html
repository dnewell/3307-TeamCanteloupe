Canteloupe Dashboard Help





How to get started


Load CSV Files

Quickly and seamlessly open a new excel CSV type file to display the data in the main window. Selecting Load Teaching *.CSV will open a file browser in which allows you to quickly choose your document.

Displaying other categories

Other categories are accessible by selecting the appropriate tabs in the dashboard. The tabs include Teaching, Publications, Funding, and Presentations.

Date Range & Filters

After selecting an excel file. Appropriate information will be displayed on the main window. The Canteloupe dash board features Date Range which will only display appropriate data from the allowed date.

Graphing

The Canteloupe dashboard offers a variety of graphs that are generated easily. Selecting the Graph with the appropriate Date Range selected. A graph will appear in a new window in which you can save.

Printing

Printing is available by selecting File > Print. You will then be prompted with a dialogue that will handle your request.

Saving your file

You can easily save your work by selecting File > Save As.
